FAQ

Частые вопросы

Всё бесплатно. Все приложения с открытым исходным кодом — скачивайте, используйте, изменяйте.
«Своя Игра» и «Миллионер» работают полностью офлайн. «ИльЯкласс» создаёт локальную сеть — интернет нужен только ученикам для подключения по QR.
Портативная — один .exe, запускается сразу с любого места (флешка, папка). Установщик — прописывает иконку в меню Пуск, ассоциации файлов. Функционально одинаковы.
Планируется. Пока все приложения собраны только под Windows (x64).
