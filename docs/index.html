02 / FAQ

Часто задаваемые вопросы

Коротко и по делу.

Всё бесплатно. Все приложения с открытым исходным кодом — скачивай, используй, изменяй как хочешь.
«Своя Игра» и «Миллионер» работают полностью офлайн. «ИльЯкласс» создаёт локальный сервер в твоей Wi-Fi сети.
Portable — один .exe, запускается с любого места (флешка, папка). Setup — установщик с иконкой в меню Пуск. Функционально одинаковы.
Планируется. Пока все приложения собраны только под Windows x64.
