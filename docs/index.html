02 / FAQ

Часто задаваемые вопросы

Коротко и по делу.

Всё бесплатно. Все приложения с открытым исходным кодом — скачивай, используй, изменяй как хочешь.
«Своя Игра» и «Миллионер» работают полностью офлайн. «ИльЯкласс» создаёт локальный сервер в Wi-Fi сети.
Portable — один .exe, запускается с любого места. Setup — установщик с иконкой в меню Пуск.
Планируется. Пока все приложения собраны только под Windows x64.
