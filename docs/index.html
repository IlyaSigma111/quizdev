FAQ

Частые вопросы

Всё бесплатно. Все приложения с открытым исходным кодом — скачивайте, используйте, изменяйте как хотите.
«Своя Игра» и «Миллионер» работают полностью офлайн — никакого интернета не нужно. «ИльЯкласс» создаёт локальный сервер в вашей Wi-Fi сети; интернет нужен только ученикам для подключения по QR-коду.
Portable — один .exe, запускается с любого места (флешка, рабочий стол, папка). Setup — установщик, прописывает иконку в меню Пуск и ассоциации файлов. По функционалу версии одинаковы.
Планируется. Пока все приложения собраны только под Windows x64.
